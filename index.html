System and Network Specialist

System and network specialists design, maintain, evaluate, and troubleshoot the hardware and software needed to connect the world via the Internet, corporate intranets, local area networks (LANs), wide area networks (WANs), and other data communication systems. Those that excel in this area are problem solvers who can assure the reliability of complex systems in a rapidly changing environment.

They work with hardware including routers, hubs, wireless adaptors, cables, and various devices, as well as software including operating systems, virtualization, networking protocols (e.g., IP/TCP), cloud computing services, network administration portals, web servers, and various security-related packages.

In this emphasis, students will combine the theory of debugging, troubleshooting, management and configuration with the reality of systems administration as they build a complete virtual model of a corporate network.

Web Developer

Web development is a broad term for the work involved in developing a web site for the Internet (World Wide Web) or an intranet (a private network). ... Among web professionals, "web development" usually refers to the main non-design aspects of building web sites: writing markup and coding.

A web developer is a programmer who specializes in, or is specifically engaged in, the development of World Wide Web applications, or applications that are run over HTTP from a web server to a web browser.

Web Designer

Web design is the process of creating websites. It encompasses several different aspects, including webpage layout, content production, and graphic design. While the terms web design and web development are often used interchangeably, web design is technically a subset of the broader category of web development.

A web designer is someone who is both creative and technically inclined, and uses both these attributes to build or redesign websites. The web designer has the ability to understand what is needed to make a website functional and easy to use, but at the same time make it aesthetically appealing to the user.
